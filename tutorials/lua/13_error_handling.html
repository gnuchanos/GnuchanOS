💜 Lesson 13: Error Handling

Lua Tutorial — GnuchanOS

Why Error Handling?

Programlar çalışırken hatalar oluşabilir: dosya bulunamaz, geçersiz input, network hatası. Lua bu hataları yakalamak ve yönetmek için araçlar sunar.

error() — Hata Fırlatma

error() fonksiyonu programı durdurur ve hata mesajı gösterir.

local function divide(a, b) if b == 0 then error("Division by zero!") end return a / b end print(divide(10, 2)) -- 5 print(divide(10, 0)) -- error: Division by zero!

assert() — Kısa Kontrol

assert() bir koşul false ise hata fırlatır.

local function sqrt(x) assert(x >= 0, "Cannot sqrt negative number") return x ^ 0.5 end print(sqrt(9)) -- 3.0 print(sqrt(-1)) -- error: Cannot sqrt negative number

pcall() — Hatayı Yakalama (Protected Call)

pcall() bir fonksiyonu güvenli şekilde çalıştırır. Hata olursa program çökmez.

local function risky(a, b) if b == 0 then error("zero!") end return a / b end local ok, result = pcall(risky, 10, 2) print(ok, result) -- true 5 ok, result = pcall(risky, 10, 0) print(ok, result) -- false tutorials/lua/13_error_handling.html:2: zero! if not ok then print("Hata yakalandi: " .. result) end
💡 pcall iki değer döndürür: (1) başarılı mı (boolean), (2) dönüş değeri veya hata mesajı

xpcall() — Gelişmiş Hata Yönetimi

xpcall() ile hata yakalayıcı (error handler) fonksiyon belirtebilirsiniz.

local function handler(err) print("Hata olustu:") print(debug.traceback(err)) end local ok, result = xpcall(risky, handler, 10, 0) -- Hata olustu: -- stack traceback: -- ... (call stack)

Practical Example: Safe File Read

local function safe_read(filename) local f, err = io.open(filename, "r") if not f then return nil, err end local content = f:read("*a") f:close() return content end local data, err = safe_read("notexist.txt") if not data then print("Dosya okunamadi: " .. err) end

Try-Catch Pattern (Lua)

Lua'da native try-catch yoktur ama pcall ile aynı desen uygulanabilir:

local function try(try_block, catch_block) local ok, err = pcall(try_block) if not ok and catch_block then catch_block(err) end end try( function() error("bir hata!") end, function(err) print("Yakalandi: " .. err) end )
