Debug Library

debug kütüphanesi ile çalışma zamanında bilgi alabilirsiniz.

-- Call stack print(debug.traceback()) -- Get info about a function local info = debug.getinfo(print) print(info.name, info.linedefined, info.source) -- Get/set local variables local x = 42 print(debug.getlocal(1, 1))

Weak Tables

Weak referanslar garbage collector'ün tablo anahtarlarını/değerlerini toplamasına izin verir.

-- Weak values: değerler GC tarafından toplanabilir local t = setmetatable({}, {__mode = "v"}) local obj = {} t[1] = obj obj = nil -- obj artık GC tarafından toplanabilir -- Weak keys: anahtarlar GC tarafından toplanabilir local cache = setmetatable({}, {__mode = "k"})

Best Practices

  • local kullanın: Her zaman local ile değişken tanımlayın
  • Error handling: Her riskli işlemde pcall kullanın
  • Module pattern: Her zaman table döndüren modül yapısı kullanın
  • self: OOP'de : (colon) notasyonu kullanın
  • Named params: Çok parametreli fonksiyonlarda table kullanın
-- İyi: local değişkenler local function greet(name) return "Hello " .. name end -- İyi: table parametre local function create_player(options) options = options or {} local name = options.name or "Player" local hp = options.hp or 100 return {name = name, hp = hp} end -- İyi: guard clauses local function divide(a, b) if b == 0 then return nil, "zero" end return a / b end
