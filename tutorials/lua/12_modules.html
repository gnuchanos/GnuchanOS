💜 Lesson 12: Modules & Standard Libs

Lua Tutorial — GnuchanOS

Modules

Lua'da modul icin dosyadan require edersin. Dosya bir table dondurur.

-- mymod.lua:
local M = {}
function M:greet(n) return "Hi " .. n end
return M
-- main.lua:
local my = require("mymod")
print(my.greet("World"))

Math Library

print(math.pi, math.sqrt(144), math.floor(4.7))
math.randomseed(os.time())
print(math.random(1, 6))

OS Library

print(os.date()) -- current date/time
print(os.time()) -- unix timestamp
print(os.clock()) -- cpu time

Debug Library

print(debug.traceback())
print(debug.getmetatable(""))
