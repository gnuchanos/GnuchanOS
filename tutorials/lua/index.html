💜 Lua Programming Language

Lightweight • Embeddable • Elegant — 14 Lessons

Lua is a powerful, efficient, lightweight scripting language designed for embedded use in applications. Created at PUC-Rio (Brazil) in 1993, it is used in games (Roblox, World of Warcraft), embedded devices, and as a configuration/extension language.
