Lua Tutorial — GnuchanOS

Syntax Rules

  • Yorumlar: -- (tek satır), --[[ ]]-- (çok satır)
  • Noktalı virgül: İsteğe bağlı (genelde kullanılmaz)
  • Bloklar: then...end, do...end ile tanımlanır
  • Case-sensitive: test ve Test farklı değişkenler
  • Global varsayılan: Değişkenler aksi belirtilmezse globaldir

Variables & Scope

x = 10 -- global

local y = 20 -- local (sadece bu blokta)

if true then
local z = 30 -- block-scoped
print(z)
end

local a, b, c = 1, "two", true
a, b = b, a -- swap
⚠️ Her zaman local kullanın — global değişkenler yanlışlıkla oluşabilir.

Global Environment (_G)

Tüm global değişkenler _G tablosunda saklanır:

x = 42
print(_G.x) -- 42
print(_G["x"]) -- 42
