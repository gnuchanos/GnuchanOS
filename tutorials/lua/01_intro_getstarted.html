Lua Tutorial — GnuchanOS

What is Lua?

Lua, 1993 yılında PUC-Rio (Brezilya) tarafından oluşturulmuş, hafif ve gömülebilir bir scripting dilidir. Adı Portekizce'de 'ay' anlamına gelir.

Kullanım alanları:

  • 🎮 Oyunlarda (Roblox, World of Warcraft, Garry's Mod)
  • 📡 Gömülü sistemlerde
  • ⚙️ Konfigürasyon/extension dili (Redis, Nginx, Wireshark)
  • 🎬 Adobe Lightroom, VLC, Awesome WM

Why Lua?

  • ✅ Hafif — ~200KB derlenmiş, milisaniyelerde başlar
  • ✅ Hızlı — LuaJIT ile C kadar hızlı olabilir
  • ✅ Gömülebilir — C/C++ projelerine eklemek çok kolay
  • ✅ Basit — sadece 21 keyword, temiz sentaks
  • ✅ Dinamik — tipler runtime'da
  • ✅ Güçlü — metatables, coroutines, first-class functions

Installation

Linux (Ubuntu/Debian):

sudo apt install lua5.4
lua -v

macOS:

brew install lua

Windows:

luabinaries.sourceforge.net — lua-5.4.x_Win64_bin.zip indir, PATH'e ekle.

Kontrol:

lua -v
Lua 5.4.x Copyright (C) 1994-2025 Lua.org, PUC-Rio

First Lua Program

hello.lua dosyası oluştur:

-- hello.lua
print("Hello, World!")

Çalıştır:

lua hello.lua
Hello, World!

Lua REPL

Terminalde lua -i veya sadece lua yaz:

Lua 5.4.x Copyright (C) 1994-2025 Lua.org, PUC-Rio
> print("Hello")
Hello
> 5 + 3
8
> os.exit()
💡 REPL çıkış: os.exit() veya Ctrl+D (Linux/macOS) / Ctrl+Z (Windows)
