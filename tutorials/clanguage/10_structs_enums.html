Lesson 10: C Structures & Enums

Intermediate

Structures (struct)

A struct groups different types of data together. Like a "record" or "row" in a table.

// Define a structure struct Student { char name[50]; int age; float gpa; }; int main() { // Create a student variable struct Student s1 = {"Alice", 20, 3.85}; // Access fields with . (dot) printf("Name: %s\n", s1.name); printf("Age: %d\n", s1.age); printf("GPA: %.2f\n", s1.gpa); return 0; }
Name: Alice
Age: 20
GPA: 3.85

typedef — Create a Short Name

typedef struct { char name[50]; int score; } Player; Player p1 = {"Mario", 9000}; // No need to write "struct Player"
💡 typedef creates an alias (nickname) for a type. Player is now a type like int.

Structs with Pointers

typedef struct { int x; int y; } Point; Point p = {10, 20}; Point* ptr = &p; printf("(%d, %d)\n", ptr->x, ptr->y); // Use -> with pointers

Nested Structures

A struct can contain another struct as a member.

typedef struct { int day; int month; int year; } Date; typedef struct { char name[50]; Date birthDate; // Nested struct float score; } Student; int main() { Student s = {"Alice", {15, 3, 2005}, 95.5}; printf("%s born %d/%d/%d\n", s.name, s.birthDate.day, s.birthDate.month, s.birthDate.year); return 0; }
Alice born 15/3/2005

Unions

A union is like a struct, but all members share the same memory. Size = largest member. Only one member can hold a value at a time.

union Data { int i; float f; char str[20]; }; int main() { union Data d; d.i = 42; printf("int: %d\n", d.i); // 42 d.f = 3.14; // Now i is overwritten! printf("float: %.2f\n", d.f); // 3.14 printf("Size: %lu bytes\n", sizeof(d)); // 20 (largest member) return 0; }
int: 42
float: 3.14
Size: 20 bytes

Struct Padding

The compiler may add padding (empty bytes) between struct members for performance. This means sizeof(struct) may be larger than the sum of its members.

#include struct Example1 { // char + int → likely 8 bytes (not 5!) char a; // 1 byte int b; // 4 bytes + 3 padding }; // Total: 8 bytes due to alignment struct Example2 { // Better order: larger types first → less padding int b; // 4 bytes char a; // 1 byte + 3 padding }; // Still 8 bytes, but better aligned printf("Example1 size: %lu\n", sizeof(struct Example1)); printf("Example2 size: %lu\n", sizeof(struct Example2));
💡 Rule of thumb: Order members from largest to smallest to minimize padding.

Enums

An enum (enumeration) gives names to integer values. Makes code more readable.

enum Weekday { MON, TUE, WED, THU, FRI, SAT, SUN }; // MON=0, TUE=1, WED=2 ... enum Weekday today = WED; printf("Day number: %d\n", today); // 2 // You can set custom values: enum Status { OK = 200, NOT_FOUND = 404, ERROR = 500 };
Day number: 2

Real-Life: Employee Database

#include #include typedef struct { int id; char name[50]; float salary; } Employee; int main() { Employee emp1 = {101, "John Doe", 55000.50}; Employee emp2; emp2.id = 102; strcpy(emp2.name, "Jane Smith"); emp2.salary = 62000.00; printf("ID: %d | Name: %s | $%.2f\n", emp1.id, emp1.name, emp1.salary); printf("ID: %d | Name: %s | $%.2f\n", emp2.id, emp2.name, emp2.salary); return 0; }
ID: 101 | Name: John Doe | $55000.50
ID: 102 | Name: Jane Smith | $62000.00
