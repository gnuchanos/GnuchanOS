Lesson 6: C Loops

Loops run code again and again until a condition is false.

While Loop

Runs while the condition is true. Check condition first.

int i = 1; while (i <= 5) { printf("%d ", i); i++; }
1 2 3 4 5
⚠️ Don't forget i++! Without it, the loop runs forever (infinite loop).

Do/While Loop

Runs at least once, then checks condition.

int x = 0; do { printf("%d ", x); x++; } while (x < 3);
0 1 2

For Loop

Best when you know exactly how many times to repeat.

// for (initialize; condition; increment) for (int i = 0; i < 5; i++) { printf("%d ", i); }
0 1 2 3 4
💡 for(int i=0; i<5; i++) = "Start at 0, run while i<5, add 1 each time"

Nested Loops

// Multiplication table 3x3 for (int i = 1; i <= 3; i++) { for (int j = 1; j <= 3; j++) { printf("%3d ", i * j); } printf("\n"); }
1 2 3
2 4 6
3 6 9

Break

break exits the loop immediately.

for (int i = 0; i < 10; i++) { if (i == 4) { break; } printf("%d ", i); }
0 1 2 3

Continue

continue skips the rest of current iteration and goes to the next.

// Print only even numbers for (int i = 0; i <= 10; i++) { if (i % 2 != 0) { continue; } printf("%d ", i); }
0 2 4 6 8 10

Real-Life: Countdown Timer

#include int main() { for (int i = 10; i > 0; i--) { printf("%d...\n", i); } printf("Blast off! 🚀\n"); return 0; }
10...
9...
8...
...
1...
Blast off! 🚀

Real-Life: PIN Checker

#include int main() { int pin = 1234, guess, attempts = 0; do { printf("Enter PIN: "); scanf("%d", &guess); attempts++; } while (guess != pin && attempts < 3); printf("%s\n", guess == pin ? "Access granted" : "Blocked"); return 0; }
