Lesson 2: C Syntax & Output

C Syntax Rules

  • Every statement ends with a semicolon ;
  • Code blocks use curly braces { }
  • C is case-sensitive — main and Main are different
#include int main() { // This is a comment - ignored by compiler printf("Hello!\n"); // ; ends the statement printf("I am learning C.\n"); return 0; }
Hello! I am learning C.

Comments

Comments are notes for humans. The compiler ignores them.

// Single-line comment /* Multi-line comment - spans several lines */

Printf() — Output to Screen

printf() prints text. The f means "formatted" — you can print text, numbers, and variables.

printf("Hello World!\n"); printf("I love C programming!\n");
Hello World! I love C programming!

Escape Sequences (Special Characters)

SequenceMeaningExample
\nNew line (Enter)printf("a\nb"); → a then b on next line
\tTab (space)printf("a\tb"); → a [tab] b
\\Print a backslashprintf("\\"); → \
\"Print a double quoteprintf("\"hi\""); → "hi"
// Escape sequences demo #include int main() { printf("Name\tAge\tCity\n"); printf("John\t25\tNYC\n"); printf("He said: \"Hello!\"\n"); printf("Path: C:\\\\Users\\\\Name\n"); return 0; }
Name Age City John 25 NYC He said: "Hello!" Path: C:\Users\Name

Statements in C

A statement is one complete command. Each statement does one job:

printf("Hello"); // Statement 1 printf("World"); // Statement 2 int x = 5; // Statement 3
⚠️ Never forget the semicolon! Every statement in C must end with ;. Missing it = compiler error.
