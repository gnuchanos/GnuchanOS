Lesson 3: C Variables & Data Types

What is a Variable?

A variable is like a box that stores data. You give it a name and a type.

// Syntax: type name = value; int age = 25; float price = 9.99; double pi = 3.1415926535; char grade = 'A';

Basic Data Types

TypeSizeFormatExample
int4 bytes%dint x = 42;
float4 bytes%ffloat f = 3.14f;
double8 bytes%lfdouble d = 3.1415926535;
char1 byte%cchar c = 'A';
short2 bytes%dshort s = 100;
long long8 bytes%lldlong long big = 99999999999LL;

Print Variables

#include int main() { int myNum = 15; float myFloat = 5.99; char myLetter = 'D'; printf("Number: %d\n", myNum); printf("Price: $%.2f\n", myFloat); printf("Grade: %c\n", myLetter); return 0; }
Number: 15
Price: $5.99
Grade: D

Change Variable Values

int score = 10; score = 20; // Changed! (no 'int' again) score = score + 5; // Now 25
⚠️ When changing, don't write the type again! ✅ score = 20; ❌ int score = 20;

Variable Naming Rules

  • ✅ Letters, digits, underscore
  • ✅ Must start with letter or underscore
  • ✅ Case-sensitive (age ≠ Age)
  • ❌ No spaces or @, #, $
  • ❌ No keywords like int, return

Booleans (True / False)

  • 0 = false
  • 1 (or any non-zero) = true
int isWeekend = 1; // true if (5) { printf("5 is true!\n"); } // Runs if (0) { printf("0 is false!\n"); } // Won't run

Type Conversion (Casting)

int x = 5; float y = x; // Implicit: 5 → 5.0 float a = 7.8; int b = (int)a; // Explicit cast: 7.8 → 7 int p = 15, q = 4; float r = (float)p / q; // 3.75 (fixes int division)

Fixed-Width Integers ()

Use when you need exact byte sizes.

#include int8_t small = 100; // 8 bits, -128 to 127 uint8_t byte = 255; // 8 bits, 0 to 255 int32_t normal = 2000000; // 32 bits

sizeof() — Memory Size

printf("int = %lu bytes\n", sizeof(int)); printf("char = %lu byte\n", sizeof(char)); printf("double = %lu bytes\n", sizeof(double));
int = 4 bytes
char = 1 byte
double = 8 bytes

Real-Life: Shopping Cart

float applePrice = 2.50; int appleCount = 3; float total = applePrice * appleCount; printf("Total: $%.2f\n", total); // $7.50
