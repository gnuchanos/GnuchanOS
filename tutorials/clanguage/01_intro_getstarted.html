🔥 Lesson 1: C Introduction & Getting Started

Simple English — W3Schools C Tutorial

What is C?

C is a programming language created in 1972 by Dennis Ritchie at Bell Labs. It is very fast and powerful. Many important programs are written in C:

  • 🖥 Operating systems (Linux, Windows, macOS)
  • 🎮 Game engines
  • 🗄 Databases (MySQL, PostgreSQL)
  • 📡 Embedded systems (cars, TVs, smart watches)

Why Learn C?

  • ✅ C teaches you how memory and computers really work
  • ✅ After C, learning C++, Java, Python is much easier
  • ✅ C code runs very fast
  • ✅ C is still in the top 5 most used languages

What You Need to Start

  1. A text editor — VS Code, Notepad++, or any code editor
  2. A C compiler — turns your code into a runnable program
CompilerInstall
GCC (recommended)Windows: MinGW. Linux: built-in (sudo apt install gcc). Mac: brew install gcc
ClangMac (built-in with Xcode). Linux: sudo apt install clang

Your First C Program

Create a file called hello.c and write this:

#include // Include standard I/O library int main() { // Program starts here printf("Hello, World!\n"); // Print to screen return 0; // Exit successfully }
Hello, World!

How to Compile and Run

# Step 1: Compile (create executable) gcc hello.c -o hello # Step 2: Run ./hello # Output: Hello, World!
💡 Every C program must have a main() function. The computer looks for main() to start execution.

Explaining Hello World

CodeWhat It Does
#include Includes the standard input/output library (for printf)
int main()Declares the main function — program entry point
{ }Curly braces define a code block
printf("...");Prints text to the screen
\nNew line character (like pressing Enter)
return 0;Tells the OS the program ended successfully
;Each statement ends with a semicolon
⚠️ Common mistake: Forgetting the semicolon ; at the end of a statement. The compiler will show an error.
