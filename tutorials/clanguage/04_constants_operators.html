Lesson 4: C Constants & Operators

Constants

A constant is a variable whose value cannot change. Use the const keyword.

const float PI = 3.14159; const int MAX_USERS = 100; // PI = 3.0; // ERROR! Can't change a constant
🔥 Tip: Use ALL_CAPS for constant names (TAX_RATE, MAX_SIZE). Easy to spot!

Real-Life: Tax Calculator

#include int main() { const float TAX = 0.18; float price = 250.00; float total = price + (price * TAX); printf("Total with tax: $%.2f\n", total); return 0; }
Total with tax: $295.00

Operators

⚠️ Integer division trap! 15 / 4 = 3 (not 3.75!). Both are int so C drops the decimal. Use float to fix.
#include int main() { int a = 15, b = 4; printf("a + b = %d\n", a + b); printf("a / b = %d (int division!)\n", a / b); float fa = 15.0, fb = 4.0; printf("fa / fb = %.2f (float division)\n", fa / fb); return 0; }
a + b = 19
a / b = 3 (int division!)
fa / fb = 3.75 (float division)

Real-Life: Roller Coaster Check

#include int main() { int age = 14; float h = 1.55; int canRide = (age >= 12) && (h >= 1.40); printf("Can ride? %s\n", canRide ? "Yes" : "No"); return 0; }
Can ride? Yes
