Lesson 8: C Functions

A function is a reusable block of code. Define it once, call it anytime.

💡 Why functions? They make code organized, reusable, and easier to debug.

Creating and Calling a Function

#include // Function declaration (prototype) - tells C the function exists void sayHello(); int main() { sayHello(); // Call the function sayHello(); // Call it again! return 0; } // Function definition void sayHello() { printf("Hello there!\n"); }
Hello there!
Hello there!

Function with Parameters and Return

// Function that takes two ints and returns their sum int add(int a, int b) { return a + b; } int main() { int result = add(5, 3); printf("5 + 3 = %d\n", result); // 8 printf("10 + 7 = %d\n", add(10, 7)); // 17 return 0; }
5 + 3 = 8
10 + 7 = 17

Pass by Value vs Pass by Reference

void changeValue(int x) { x = 100; // Only changes COPY } void changeRef(int *ptr) { *ptr = 100; // Changes ORIGINAL! } int main() { int num = 5; changeValue(num); printf("After changeValue: %d\n", num); // Still 5 changeRef(&num); printf("After changeRef: %d\n", num); // Now 100 return 0; }
After changeValue: 5
After changeRef: 100
💡 Pass by value (default): function gets a copy. Pass by reference (with pointers): function can modify the original.

Inline Functions

inline tells the compiler to copy the function code directly where it's called (no jump overhead). Useful for very small, fast functions.

#include inline int max(int a, int b) { return (a > b) ? a : b; } int main() { printf("%d\n", max(10, 20)); // 20 - code is "inlined" here return 0; }
💡 inline is a hint to the compiler — it may ignore it for large functions. Use for tiny 1-2 line functions.

Callback Functions (Function Pointers as Parameters)

Pass a function pointer as an argument to another function. Used in sorting, event handling, etc.

#include // Function that takes a callback void processArray(int arr[], int size, int (*callback)(int)) { for (int i = 0; i < size; i++) { arr[i] = callback(arr[i]); } } // Callback functions int doubleIt(int x) { return x * 2; } int squareIt(int x) { return x * x; } int main() { int nums[] = {1, 2, 3, 4}; int len = 4; processArray(nums, len, doubleIt); // Now: {2, 4, 6, 8} for (int i = 0; i < len; i++) printf("%d ", nums[i]); printf("\n"); processArray(nums, len, squareIt); // Now: {4, 16, 36, 64} for (int i = 0; i < len; i++) printf("%d ", nums[i]); printf("\n"); return 0; }
2 4 6 8
4 16 36 64

Scope (Where Variables Live)

int global = 999; // Global - accessible everywhere void test() { int local = 50; // Local - only in this function } int main() { int x = 10; // Local to main() // x is NOT visible in test() // local is NOT visible here printf("%d\n", global); // OK - global works everywhere return 0; }

Real-Life: Calculator with Functions

#include int square(int n) { return n * n; } int cube(int n) { return n * n * n; } int main() { int num = 5; printf("%d squared = %d\n", num, square(num)); printf("%d cubed = %d\n", num, cube(num)); return 0; }
5 squared = 25
5 cubed = 125
