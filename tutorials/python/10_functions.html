Lesson 10: Functions

Simple English — Based on W3Schools Python Tutorial

Defining a Function (def)

A function is a block of code that runs only when it is called. In Python we use def to define a function.

def greet(): print("Hello, World!") greet()
Hello, World!

Parameters vs Arguments

Parameters are the variables listed inside the parentheses in the function definition. Arguments are the values you send into the function when you call it.

def greet(name): # name is a parameter print(f"Hello, {name}!") greet("Alice") # "Alice" is an argument
Hello, Alice!

Return Statement

Use return to send a value back from a function. Without return, the function returns None.

def add(a, b): return a + b result = add(5, 3) print(result) def no_return(): print("This function returns None") print(no_return())
8 This function returns None None

Default Parameters

You can give a parameter a default value. If no argument is passed, the default is used.

def greet(name="Guest"): print(f"Hello, {name}!") greet() greet("Bob")
Hello, Guest! Hello, Bob!

Keyword Arguments

When calling a function, you can specify arguments by their parameter name. This lets you pass them in any order.

def display(name, age): print(f"{name} is {age} years old") display(age=25, name="Alice") # keyword arguments
Alice is 25 years old

*args – Variable Positional Arguments

*args allows a function to accept any number of positional arguments. They become a tuple inside the function.

def sum_all(*args): return sum(args) print(sum_all(1, 2, 3, 4)) print(sum_all(10, 20))
10 30

**kwargs – Variable Keyword Arguments

**kwargs allows a function to accept any number of keyword arguments. They become a dict inside the function.

def print_info(**kwargs): for key, value in kwargs.items(): print(f"{key}: {value}") print_info(name="Alice", age=30, city="London")
name: Alice age: 30 city: London

Lambda Functions

A lambda is a small anonymous function defined in one line. It can take any number of arguments but has only one expression.

# Normal function def square(x): return x ** 2 # Lambda equivalent square_lambda = lambda x: x ** 2 print(square(5)) print(square_lambda(5)) # Using lambda with map() numbers = [1, 2, 3, 4] squared = list(map(lambda x: x**2, numbers)) print(squared)
25 25 [1, 4, 9, 16]

Scope – LEGB Rule

Python looks for variables in this order: Local → Enclosing → Global → Built-in.

x = "global" # Global scope def outer(): x = "enclosing" # Enclosing scope def inner(): x = "local" # Local scope print("Inner:", x) inner() print("Outer:", x) outer() print("Global:", x)
Inner: local Outer: enclosing Global: global

Global Keyword

The global keyword lets you modify a global variable from inside a function.

counter = 0 def increment(): global counter counter += 1 increment() increment() print(counter)
2

Nonlocal Keyword

nonlocal lets you modify a variable from the enclosing (outer) scope inside a nested function.

def outer(): count = 0 def inner(): nonlocal count count += 1 return count return inner counter_fn = outer() print(counter_fn()) print(counter_fn()) print(counter_fn())
1 2 3

Docstrings

A docstring is a string literal that appears right after a function definition, used to document what the function does.

def add(a, b): """Return the sum of a and b.""" return a + b print(add.__doc__) print(help(add))
Return the sum of a and b. Help on function add in module __main__: add(a, b) Return the sum of a and b.

Type Hints

Type hints indicate the expected type of parameters and return values. They don't enforce types but help with readability and tools.

def greet(name: str) -> str: return f"Hello, {name}!" result: str = greet("Alice") print(result)
Hello, Alice!

Recursion

Recursion is when a function calls itself. Every recursive function needs a base case to stop.

def factorial(n): """Calculate factorial using recursion.""" if n == 1: # base case return 1 return n * factorial(n - 1) print(factorial(5)) print(factorial(7))
120 5040

Nested Functions

You can define a function inside another function. The inner function is only available inside the outer function.

def outer(x): def inner(y): return y * 2 return inner(x) + 1 print(outer(5)) print(outer(10))
11 21
Summary: Functions help you organize code into reusable blocks. You can pass data via parameters, return results, use lambdas for quick functions, control variable scope, and even have functions call themselves with recursion.
