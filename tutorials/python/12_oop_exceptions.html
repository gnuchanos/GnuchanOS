Lesson 12: OOP & Exceptions

Simple English — Based on W3Schools Python Tutorial

Classes & Objects

class Dog: def __init__(self, name, age): self.name = name self.age = age def bark(self): print(f"{self.name} says Woof!") d = Dog("Rex", 3) print(d.name) d.bark()

Inheritance

class Animal: def __init__(self, name): self.name = name class Cat(Animal): # Cat inherits from Animal def meow(self): print(f"{self.name} says Meow!") c = Cat("Whiskers") c.meow()

try/except/finally

try: x = 10 / 0 # This raises ZeroDivisionError except ZeroDivisionError: print("Cannot divide by zero!") finally: print("This always runs.")

Custom Exceptions

try: age = -5 if age < 0: raise ValueError("Age cannot be negative!") except ValueError as e: print(e)
💡 Always catch specific exceptions (not bare except:). Use finally for cleanup.
