Lesson 7: Strings

Simple English — Based on W3Schools Python Tutorial

Creating Strings

You can create strings with single quotes, double quotes, or triple quotes:

s1 = 'Hello' s2 = "World" s3 = """Multi line string""" print(s1) print(s2) print(s3)
Hello World Multi line string

String Indexing

Each character has an index. Index starts at 0. Negative index counts from the end.

s = "Python" print(f"First char: {s[0]}") print(f"Third char: {s[2]}") print(f"Last char: {s[-1]}") print(f"Second from end: {s[-2]}")
First char: P Third char: t Last char: n Second from end: o

Slicing

s[start:end:step] — extracts a part of the string:

s = "Python Programming" print(s[0:6]) # "Python" print(s[7:]) # "Programming" print(s[:6]) # "Python" print(s[::2]) # every 2nd character print(s[::-1]) # reverse the string
Python Programming Python Pto rgamn gnimmargorP nohtyP

Strings Are Immutable

⚠ You cannot change a character by index. This will raise an error:
s = "Hello" # s[0] = "J" # ❌ TypeError: 'str' object does not support item assignment s = "J" + s[1:] # ✅ Create a new string instead print(s)
Jello

String Methods

upper() and lower()

msg = "Hello World" print(msg.upper()) print(msg.lower())
HELLO WORLD hello world

strip()

Removes whitespace from both ends:

txt = " Hello! " print(f"'{txt.strip()}'")
'Hello!'

replace()

txt = "I like apples" print(txt.replace("apples", "bananas"))
I like bananas

split()

csv = "apple,banana,cherry" items = csv.split(",") print(items)
['apple', 'banana', 'cherry']

join()

words = ["Python", "is", "fun"] sentence = " ".join(words) print(sentence)
Python is fun

find()

Returns the index of the first occurrence, or -1 if not found:

txt = "Hello, welcome to Python" print(txt.find("welcome")) print(txt.find("Java"))
7 -1

count()

txt = "banana" print(txt.count("a")) print(txt.count("na"))
3 2

startswith() and endswith()

file = "image.png" print(file.startswith("image")) print(file.endswith(".png")) print(file.endswith(".jpg"))
True True False

f-Strings (Formatted Strings)

name = "Alice" age = 25 price = 49.9567 print(f"Name: {name}, Age: {age}") print(f"Price: {price:.2f}") # 2 decimal places print(f"|{name:>10}|") # right-aligned in 10 spaces print(f"|{name:<10}|") # left-aligned print(f"|{name:^10}|") # centered
Name: Alice, Age: 25 Price: 49.96 | Alice| |Alice | | Alice |

len() — String Length

s = "Hello" print(f"Length: {len(s)}")
Length: 5

in / not in (Membership)

txt = "The rain in Spain" print("rain" in txt) print("snow" not in txt)
True True

Escape Sequences

print("Line1\\nLine2") # newline print("Tab\\tseparated") # tab print("Backslash: \\\") # backslash print("Quote: \"") # double quote
Line1 Line2 Tab separated Backslash: \ Quote: "

Raw Strings

Raw strings treat backslashes as literal characters — useful for regex and file paths:

path = r"C:\\Users\\Name" regex = r"\\d+" # matches one or more digits print(path) print(regex)
C:\Users\Name \d+

String Concatenation

a = "Hello" b = "World" c = a + " " + b print(c) print(a * 3) # repeat 3 times
Hello World HelloHelloHello

chr() and ord()

ord() returns the Unicode code point of a character. chr() converts a code point back to a character.

print(ord("A")) # 65 print(chr(65)) # "A" print(ord("ñ")) # 241 print(chr(241)) # "ñ"
65 A 241 ñ
💡 Strings are immutable — every method returns a new string. The original stays unchanged.
💡 Use f-strings for cleaner, faster string formatting. They work in Python 3.6+.
