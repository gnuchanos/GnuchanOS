Lesson 2: Python Syntax & Output

Simple English — Based on W3Schools Python Tutorial

Python Syntax Rules

  • No semicolons needed — just write the statement
  • Indentation matters — spaces/tabs define code blocks
  • Comments start with #

Basic print()

print("Hello, World!") print(42) print(3.14) print("Hello", "World") # Comma adds a space
Hello, World! 42 3.14 Hello World

Comments

# This is a single-line comment """ This is a multi-line comment (docstring) """

Escape Sequences

SequenceMeaning
\nNew line
\tTab
\\Backslash
\"Double quote
\'Single quote
print("Line 1\nLine 2") print("Name:\tAge") print("He said: \"Hi!\"")
Line 1 Line 2 Name: Age He said: "Hi!"

The sep and end Parameters

print("a", "b", "c", sep="-") print("Hello", end=" ") print("World")
a-b-c Hello World
💡 Indentation is how Python knows which code belongs together. Use 4 spaces for each level.
