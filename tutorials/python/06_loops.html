Lesson 6: Loops

Simple English — Based on W3Schools Python Tutorial

for Loop

A for loop iterates over each item in a sequence (list, string, range, etc.).

fruits = ["apple", "banana", "cherry"] for fruit in fruits: print(fruit)
apple banana cherry

while Loop

Repeats as long as a condition remains True. Be careful — it can run forever!

count = 0 while count < 5: print(count, end=" ") count += 1 print("\nDone!")
0 1 2 3 4 Done!

break — Exit the Loop

break immediately stops the loop, no matter what.

for i in range(1, 10): if i == 5: break print(i, end=" ") print("\nLoop stopped at 5")
1 2 3 4 Loop stopped at 5

continue — Skip to Next Iteration

continue skips the rest of the current iteration and moves to the next one.

for i in range(1, 8): if i % 2 == 0: continue print(i, end=" ") print("\nOnly odd numbers")
1 3 5 7 Only odd numbers

else in Loops

Both for and while can have an else block. It runs only if the loop completed normally (no break).

# else runs — no break for i in range(3): print(i) else: print("Loop finished, no break!") print("---") # else does NOT run — break hit for i in range(3): if i == 1: break print(i) else: print("Won't run because of break")
0 1 2 Loop finished, no break! --- 0
💡 Loop else: The else block runs only if the loop ended normally (not via break). It's like a "no-break" reward block.

Nested Loops

You can put one loop inside another. The inner loop runs completely for each outer iteration.

for row in range(1, 4): for col in range(1, 4): print(row * col, end="\t") print() # new row
1 2 3 2 4 6 3 6 9

enumerate() — Loop with Index

When you need both the index and the value in a for loop, use enumerate().

colors = ["red", "green", "blue"] for index, color in enumerate(colors): print(f"{index}: {color}")
0: red 1: green 2: blue

zip() — Parallel Iteration

Loop over multiple sequences at the same time with zip().

names = ["Alice", "Bob", "Charlie"] scores = [85, 92, 78] grades = ["B", "A", "C"] for name, score, grade in zip(names, scores, grades): print(f"{name}: {score} → {grade}")
Alice: 85 → B Bob: 92 → A Charlie: 78 → C

pass vs continue

  • pass — does nothing, used as placeholder (syntax needs a statement)
  • continue — skips current iteration and goes to next
# pass — just placeholder for i in range(3): if i == 1: pass # Does nothing, continues normally print(i, end=" ") print("\n---") # continue — skips the rest for i in range(3): if i == 1: continue # Skips printing 1 print(i, end=" ") print()
0 1 2 --- 0 2

Infinite Loop — while True

Sometimes you want a loop that runs forever until a condition inside breaks it.

attempts = 0 while True: attempts += 1 print(f"Attempt {attempts}") if attempts >= 3: print("Maximum attempts reached") break
Attempt 1 Attempt 2 Attempt 3 Maximum attempts reached
⚠️ Be careful with infinite loops! If you forget to add a break, your program will run forever. Always ensure there's a way out.

Putting It All Together

numbers = [7, 12, 5, 18, 3, 9] print("Processing numbers:") for idx, num in enumerate(numbers): if num % 2 == 0: continue # skip even numbers if num > 15: print("Found a big number, stopping!") break print(f" [{idx}] {num} is odd") else: print("All odd numbers processed (no break)")
