What is Python?

Python is a programming language created in 1991 by Guido van Rossum. It is known for being:

  • ✅ Easy to read — clean, simple syntax that looks like plain English
  • ✅ Interpreted — you run code directly, no compilation step needed
  • ✅ Versatile — web development, data science, AI, automation, games, and more
  • ✅ Huge community — thousands of free libraries and frameworks
  • ✅ Cross-platform — works on Windows, macOS, Linux, even Raspberry Pi
💡 Python is named after the comedy group Monty Python, not the snake! The logo shows two snakes because of this.

Why Learn Python?

  • 🚀 #1 language for beginners — easiest to learn
  • 📊 Used in data science & machine learning (NumPy, Pandas, TensorFlow)
  • 🌐 Powers websites like YouTube, Instagram, and Spotify
  • 🤖 Great for automation & scripting
  • 💼 High demand in the job market with great salaries
  • 🎮 Game development with PyGame

Your First Python Program

Let's write your very first Python program! Create a file called hello.py and write this inside:

print("Hello, World!")

How to Run It

Open a terminal / command prompt in the same folder and type:

# On macOS / Linux: python3 hello.py # On Windows: python hello.py

Output:

Hello, World!
💡 print() is Python's built-in function that displays output. The text inside quotes is a string. The parentheses () are required in Python 3.

Python REPL (Interactive Shell)

You can also run Python interactively without creating a file. Just type python (or python3) in your terminal:

>>> print("Hello, World!") Hello, World! >>> 5 + 3 8 >>> exit()

The >>> prompt means Python is waiting for your command. Type exit() or press Ctrl+Z (Windows) or Ctrl+D (Mac/Linux) to leave.

Python 3.x.x (default, ...) Type "help", "copyright", "credits" or "license" for more information. >>> print("Hello, World!") Hello, World! >>> 5 + 3 8 >>> exit()
💡 The REPL is great for testing small pieces of code quickly. You don't need to create a file — just type and see results immediately!

Case Sensitivity

Python is case-sensitive. This means uppercase and lowercase letters are different:

print("Hello") # ✅ Works — lowercase 'print' Print("Hello") # ❌ Error! Python doesn't know 'Print'
Hello Traceback (most recent call last): File "", line 1, in NameError: name 'Print' is not defined
⚠️ Always use the correct case for Python keywords, functions, and variable names. print works, Print does not!

Python File Extensions

Python files end with .py extension. You run them from the terminal:

# Create a file named hello.py # Write Python code inside # Then run it: python hello.py
💡 You can also create .pyw files on Windows — they run without opening a terminal window. Useful for GUI programs.

Multi-line Strings with Triple Quotes

You can create strings that span multiple lines using triple quotes (""" or '''):

poem = """Roses are red, Violets are blue, Python is fun, And so are you!""" print(poem)
Roses are red, Violets are blue, Python is fun, And so are you!
# Triple single quotes work too: text = '''This is a multi-line string.''' print(text)
This is a multi-line string.

Comments

Comments are notes you add to your code that Python ignores. They help you (and others) understand the code later.

Single-line comments with #

# This is a comment — Python ignores everything after # print("Hello") # Comments can be at the end of a line too # print("This won't run") # You can "comment out" code temporarily
Hello

Multi-line comments (docstrings)

Triple quotes can also be used as docstrings — multi-line comments that document your code:

""" This is a docstring. It can span multiple lines. Python uses this as documentation. """ # This also works with single quotes: '''Another way to write a multi-line comment.''' print("Comments above are ignored")
Comments above are ignored
💡 Use comments to explain why you wrote code a certain way, not what the code does (the code itself should be clear enough). Good comments save hours of confusion later!
