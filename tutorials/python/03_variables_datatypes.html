Lesson 3: Variables & Data Types

Simple English — Based on W3Schools Python Tutorial

Variables

Variables store data. No int x or let x needed — just assign!

name = "Alice" age = 25 height = 1.75 is_student = True print(name) print(age + 5)
Alice 30

Variable Names

  • Letters, numbers, underscores — start with letter or underscore
  • Case-sensitive: age ≠ Age
  • No keywords: if, for, while, etc.

Basic Data Types

TypeExample
int5, -3, 0
float3.14, -0.5
str"hi", 'bye'
boolTrue, False
NoneTypeNone

Type Conversion

x = "5" y = 2 print(int(x) + y) # 7 print(float("3.14")) # 3.14 print(str(42)) # "42"
7 3.14 42

f-Strings

name = "Bob" age = 30 print(f"My name is {name} and I am {age}.") print(f"Next year: {age + 1}")
My name is Bob and I am 30. Next year: 31
💡 f-strings use f"text {expr}". Works in Python 3.6+.
