Lesson 5: Conditions (if/elif/else)

Simple English — Based on W3Schools Python Tutorial

The if Statement

An if statement runs a block of code only when a condition is True.

x = 10 if x > 0: print("x is positive")
x is positive
⚠️ Don't forget the colon (:) at the end of the condition! Everything inside the if block must be indented (4 spaces).

elif — Multiple Conditions

Check several conditions in order. The first True condition runs its block; the rest are skipped.

score = 85 if score >= 90: print("Grade: A") elif score >= 80: print("Grade: B") elif score >= 70: print("Grade: C") else: print("Grade: F")
Grade: B

else — Catch All

else runs when no condition is True.

age = 17 if age >= 18: print("You can vote") else: print("Too young to vote")
Too young to vote

Nested if (İç İçe)

You can put an if inside another if.

num = 15 if num > 0: print("Positive number") if num % 2 == 0: print("and even") else: print("and odd") else: print("Non-positive")
Positive number and odd

Ternary Operator (Conditional Expression)

A short way to write if-else in one line: x if condition else y.

age = 20 status = "Adult" if age >= 18 else "Minor" print(status) num = 7 result = "Even" if num % 2 == 0 else "Odd" print(f"{num} is {result}")
Adult 7 is Odd

Truthy & Falsy Values

In Python, some values are considered False in a condition without being explicitly False. These are called falsy values.

FalsyTruthy
0, 0.0Any non-zero number
"" (empty string)Any non-empty string
[], (), {} (empty collections)Non-empty collections
NoneEverything else
FalseTrue
if 0: print("Won't print — 0 is falsy") if "hello": print("Will print — non-empty string is truthy") data = [] if not data: print("Empty list — falsy") value = None if value is None: print("value is None")
Will print — non-empty string is truthy Empty list — falsy value is None
💡 Use if x: to check if a value is "truthy" (non-zero, non-empty, not None). Use if not x: to check if it's "falsy".

Short-Circuit with and / or

Python stops evaluating as soon as the result is determined. Use this for concise condition chains.

user = "Alice" password = "secret" # 'and' short-circuits on first False if user and password: print("Both user and password exist") # 'or' short-circuits on first True name = "" display = name or "Guest" print(f"Hello, {display}")
Both user and password exist Hello, Guest

match-case (Python 3.10+)

A cleaner alternative to many elif branches. Similar to switch in other languages.

command = "start" match command: case "start": print("Starting...") case "stop": print("Stopping...") case "restart": print("Restarting...") case _: print("Unknown command") # You can also match patterns with values point = (0, 5) match point: case (0, y): print(f"On the Y axis at {y}") case (x, 0): print(f"On the X axis at {x}") case (x, y): print(f"At ({x}, {y})")
Starting... On the Y axis at 5

pass — Empty Block

If you need an if or else block that does nothing (syntax placeholder), use pass.

x = 42 if x > 100: pass # TODO: handle this case later else: print(f"x is {x}")
x is 42

Putting It All Together

temperature = 30 weather = "sunny" if temperature > 25 and weather == "sunny": print("It's a hot sunny day! 🏖️") elif temperature > 15: print("Pleasant weather") else: print("Better stay inside") # Ternary example activity = "go out" if temperature > 20 else "stay home" print(f"Suggestion: {activity}")
It's a hot sunny day! 🏖️ Suggestion: go out
